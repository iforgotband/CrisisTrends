# Quickstart

1. Fork this repository.
1. `git clone https://github.com/YourUserName/CrisisTrends`
1. `npm install -g gulp outdated`
1. `cd CrisisTrends`
1. `npm install`
1. `gulp serve`
1. Start coding!
1. `git checkout branch depends/bump/minor/gulp` (or whatever you're doing, descriptively)
1. `gulp test`
1. `gulp serve`
